# Spark Fantasy Console

Spark es una fantasy console moderna pero retro-inspirada. Su objetivo es ofrecer un entorno simple, limitado y creativo para desarrollar juegos.

## Características

* CPU virtual con ciclos limitados
* 64 KB de RAM
* Resolución fija de 160x160
* Sprites, tilemap, sonido y música
* Cartuchos auto-contenidos
* Lenguaje propio (SparkLang)

## Flujo de Trabajo

1. Abrir Spark Editor
2. Crear o cargar un cartucho
3. Editar:

   * Código (SparkLang)
   * Sprites
   * Mapas
   * Sonidos
4. Ejecutar la consola virtual
5. Exportar cartucho

## Requisitos

* Python 3.11+

## Ejecución

```
python spark.py cartucho.sparkcart
```
